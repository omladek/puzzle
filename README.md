## Usage

Make sure you have [Node](https://nodejs.org) installed.

Install gulp globally: `npm install -g gulp`.

### Development

Run `gulp --dev` and develop happily in `src` folder.

### Production build

Run `gulp` and pick built assets from `dist` folder.
